1. üè† System Overview

The Air Monitoring System is a comprehensive web-based application designed for managing air quality monitoring projects, client relationships, timesheets, and invoicing. The system provides role-based access control and integrates with Xero for accounting purposes.

Key Features:

  • Project Management: Track air monitoring projects from creation to completion
  • Client Management: Maintain client information and project history
  • Timesheet Tracking: Record and manage work hours for projects
  • Invoice Management: Create, edit, and track invoices with Xero integration
  • Admin Controls: Manage users, custom data fields, and system settings
  • Dashboard: Real-time overview of project statuses and key metrics

4. üìÅ Projects

Projects are the core of the system, representing individual air monitoring assignments from clients.

4.1 Projects Overview

The projects page displays all projects in a sortable, filterable table with the following information:

  • Project ID: Unique identifier (e.g., LDJ05116)
  • Name: Project title/description
  • Client: Associated client
  • Status: Current project status with color coding
  • Users: Assigned team members
  • Created Date: When the project was created

Search and Filter

To search projects:
  1. Use the search bar to find projects by name, client, or project ID
  2. Filter by status using the status dropdown
  3. Sort by any column by clicking the column header

4.2 Adding New Projects

To create a new project:
  1. Click the "ADD PROJECT" button
  2. Fill in the required information:
    • Project Name: Descriptive title for the project
    • Client: Select from existing clients or add new
    • Department: Choose from Asbestos & HAZMAT, Lead, Mould, etc.
    • Status: Defaults to "In progress"
    • Project Type: Regular or Large project
  3. Add optional details:
    • Work Order: Client's work order number
    • Location: Project site address
    • Project Contact: On-site contact information
    • Notes: Additional project information
  4. Click "Create Project"

Project ID Generation: The system automatically generates a unique project ID (e.g., LDJ05116) based on the project type and existing projects. This ID is used for tracking and reference throughout the project lifecycle.

4.3 Editing Projects

To edit an existing project:
  1. Click on the project name or use the "Edit" button
  2. Modify any of the project details
  3. Update assigned users if needed
  4. Click "Update Project" to save changes

Unsaved Changes Warning: If you navigate away from a project with unsaved changes, you'll be prompted to save or discard your changes.

6. üí∞ Invoices

The invoices module handles all billing activities, including invoice creation, editing, and Xero integration.

6.1 Invoices Overview

The invoices page displays all invoices with the following information:

  • Invoice Number: Unique invoice identifier
  • Project ID: Associated project
  • Client: Billing client
  • Amount: Invoice total
  • Date: Invoice date
  • Due Date: Payment due date
  • Status: Current invoice status

6.2 Creating Invoices

To create a new invoice:
  1. Click the "Add Invoice" button
  2. Select a project (this auto-populates client information)
  3. Enter invoice details:
    • Invoice Number: Unique identifier
    • Amount: Total invoice amount
    • Date: Invoice date
    • Due Date: Payment due date (auto-calculated based on client terms)
    • Description: Invoice description
    • Reference: Additional reference information
  4. Click "Create Invoice" to save

6.3 Draft Invoices

Draft invoices allow you to create detailed invoices with line items before finalizing them.

To create a draft invoice:
  1. Navigate to the Draft Invoices section
  2. Click "Create Draft Invoice"
  3. Select a project and client
  4. Add invoice line items:
    • Description: Item description
    • Quantity: Number of units
    • Rate: Price per unit
    • Account: Chart of accounts code
    • Tax Rate: Tax classification
  5. Review totals and save as draft

6.4 Xero Integration

Xero Integration: The system integrates with Xero accounting software for seamless invoice management and payment tracking.

To sync with Xero:
  1. Ensure you have Xero sync permissions
  2. Click the "Sync with Xero" button
  3. Select invoices to sync
  4. Review sync status and resolve any errors
Note: Xero integration requires specific permissions. Contact your administrator if you don't see Xero options.

9. üîß Troubleshooting

Common Issues

Page Loading Slowly

Solution: Check your internet connection and try refreshing the page. Large datasets may take time to load.

Can't Save Changes

Possible Causes:

  • You don't have permission to edit the item
  • Another user is editing the same item
  • Required fields are missing
  • Data validation errors

Solution: Check error messages, ensure all required fields are filled, and verify your permissions.

Search Not Working

Solution: Try clearing the search and typing again. Some searches may take a moment to process large datasets.

Timesheet Entry Issues

Common Problems:

  • Cannot edit finalized timesheet entries
  • Time entries not saving
  • Project not appearing in dropdown

Solutions:

  • Contact admin to unfinalize timesheet if needed
  • Ensure all required fields are completed
  • Check that the project is in an active status

Getting Help

If you encounter issues not covered in this manual:

  1. Check the error messages displayed in the application
  2. Try refreshing the page and attempting the action again
  3. Contact your system administrator
  4. Document the steps that led to the issue for better support
